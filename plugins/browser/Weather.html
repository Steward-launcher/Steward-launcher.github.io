Weather

命令

查看天气

trigger: tq

操作

  • query 不存在时，查询上一次查询过的城市的天气, 如果没有，则查询北京的
  • query 存在时，将 query 作为城市名进行天气查询

使用的百度天气 api，免费版，有使用次数限制，如果捐助多的话可以换别的~
