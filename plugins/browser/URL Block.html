URL Block

命令

屏蔽网站

trigger: bk

  • 批量操作支持

操作

  • query 为空时，显示的是已屏蔽的网站列表
    • Enter 删除选定的网站屏蔽
  • query 不为空时，将 query 部分作为网址添加到屏蔽列表中
    • Enter 添加到屏蔽列表

屏蔽网站 8 小时

除了强制屏蔽网站 8 小时后才能解除，其它跟 bk 一致
